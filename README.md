# frontend_pakages

Hirediversity frontend 개발을 위한 정적 자료, api, React 컴포넌트

## 사용방법

'''
cd {your frontend directory}/src
git submodule add {link of this repository} pakages
'''

## 제공기능

assets : 정적자료

- datas : 언어자료
- images : 사진자료

apis : api

components : React 컴포넌트

- FormUI : Hirediversity 자체폼 컴포넌트
